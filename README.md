# Adopt-A-Stray
Τελική εργασία ΙΕΚ  - Πρακτικη Εφαρμογή στην Ειδικότητα


Η εργασία είχε ως θέμα την κατασκευή του front end μιας εταιρείας.Έκανα το design σε Figma  και το front end με HTML, CSS και Javascript για να είναι responsive το navigation.Οι σελίδες είναι responsive.Τα δεδομένα έχουν τοποθετηθεί στατικά το οποίο εν πρόκειτο να αλλάξει όταν μάθω την php σε καλύτερο βαθμό.
